<Note>
Use incident.io with Microsoft Teams to manage incidents directly in your team's primary communication platform.
</Note>

## Overview

The incident.io integration for Microsoft Teams brings powerful incident management capabilities directly into your workspace. Declare incidents, coordinate responses, and keep everyone informed without leaving Teams.

## Key Features

<CardGroup cols={2}>
  <Card title="Declare Incidents" icon="exclamation-triangle">
    Start an incident response directly from Microsoft Teams with a simple command.
  </Card>
  <Card title="Channel Management" icon="hashtag">
    Automatically create and manage incident channels for focused collaboration.
  </Card>
  <Card title="Notifications" icon="bell">
    Receive real-time updates about incident status and escalations.
  </Card>
  <Card title="Private Incidents" icon="lock">
    Handle sensitive incidents with private channel support.
  </Card>
</CardGroup>
